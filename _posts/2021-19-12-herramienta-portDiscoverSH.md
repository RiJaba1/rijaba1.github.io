## Utilidad
Esta herramienta está diseñada para poder hacer un escaneo rápido y poco ruidoso a una IP que nosotros le asignemos.

El enlace a GitHub para descargar esta utilidad lo teneis más abajo.

## Instalación
La instalación de esta herramienta es muy sencilla dado que usa herramientas que ya vienen instaladas de manera predeterminada en la mayoría de Linux.

```
git clone https://github.com/RiJaba1/portDiscoverSH/
cd portDiscoverSH
sudo chmod +x portDiscover.sh
```

## Uso
El uso de esta herramienta es también secillo dado que solo necesitas ejecutarla con `./portDiscover.sh` y ya te guiará en todos los pasos.
